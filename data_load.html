2  Data Loading

In this section, we load our dataset and take an initial look at its structure. This first inspection step is important for understanding what we are working with before doing any cleaning or modeling.

2.1 Load the Data

df_raw <- read_csv("data/penguins.csv", show_col_types = FALSE)

2.3 Initial Observations

From this first look, we can already identify some issues that will need to be addressed:

  • some body measurements have missing values
  • categorical variables may need to be recoded as factors
  • we can create a binary outcome for classification

These observations will guide the data cleaning step in the next section.
